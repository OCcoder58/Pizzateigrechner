# Pizzateig-Rechner

Das ist ein einfacher Pizzateigrechner, den ich mit TypeScript umgesetzt habe.

Man kann die gewünschte Mehlmenge eingeben und bekommt automatisch die passenden Mengen für Wasser, Salz, Zucker, Olivenöl und Hefe berechnet.

Das Rezept basiert auf meinem eigenen New York Slice Teig.

## Verwendung

1. Mehlmenge eingeben  
2. Auf „Berechnen“ klicken  
3. Zutaten werden direkt angezeigt  

## Technik

- TypeScript (src → dist)
- HTML & CSS
